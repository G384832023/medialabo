各授業回の学習事項

第1回 WWW と HTML

第1回の授業ではhtmlを利用した簡単なWebページの作成方法です。要素について学び、それをうまく組み合わせて演習1-4のようなちょっとしたページを作ることができた。要素にはたくさんの種類があり少しずつ覚えていく必要がある。書いてある文字をリンクにする要素があるのだが、どのサイトからどのサイトに飛ばすかによって要素の書き方が変わってくるので難しい。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回の授業では作成したhtmlのページを装飾するcssを学びました。それぞれのページをリンクで繋げ、cssを書くことでそれに応じてhtmlに装飾が追加されていく。cssはセレクタやプロパティ・プロパティ名の3つの要素で構成されていてたくさんの装飾種類がある。

演習の成果: 練習問題2-3: ウェブブラウザ説明ページの装飾
