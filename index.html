各授業回の学習事項

第1回 WWW と HTML

第1回の授業ではhtmlを利用した簡単なWebページの作成方法です。要素について学び、それをうまく組み合わせて演習1-4のようなちょっとしたページを作ることができた。要素にはたくさんの種類があり少しずつ覚えていく必要がある。書いてある文字をリンクにする要素があるのだが、どのサイトからどのサイトに飛ばすかによって要素の書き方が変わってくるので難しい。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回の授業では作成したhtmlのページを装飾するcssを学びました。それぞれのページをリンクで繋げ、cssを書くことでそれに応じてhtmlに装飾が追加されていく。cssはセレクタやプロパティ・プロパティ名の3つの要素で構成されていてたくさんの装飾種類がある。

演習の成果: 練習問題2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回の授業ではWebページ上で複雑な機能を実現するプログラミング言語であるJavaScriptについて学びました。前回学んだページを装飾するcssに加えてページに動きをつけるものがJavaScriptであり、普段私たちが使っているJavaとは少しプログラムの書き方に違いがある。

演習の成果: 練習問題3-6: オブジェクトの参照

第4回 DOM と JavaScript

DOMとはWEBコンテンツを表現するデータ構造のことであり、DOMとJavaScriptを組み合わせることによって動きのあるWEBページを作ることができる。この授業ではDOMを使って元々あるHTMLに要素を追加・削除・cssの書き換えなどを行いました。直接htmlを書き換えればいいのでは、と思ったが仕事としてプログラムを書いた時にすごく楽になることがわかりました。

演習の成果: 練習問題4-2~4-5: DOMプログラミング

第5回 イベント処理

主題であるイベントとはWEBページ上で起こった動作や出来事のことである。今回のイベント処理の講義ではボタンを設置してそのページで検索を行うことのできるプログラムを作りました。イベント処理が実行される要素にもたくさんの種類がある。検索するものの要素にも色や時間・電話番号などさまざまな種類がある。

演習の成果: 練習問題5-3: イベントを選択する
